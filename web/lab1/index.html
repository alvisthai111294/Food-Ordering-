About HTML

HTML(Hypertext Markup Language is the language used to create webpages. "Hypertext" refers to the hyperlinks that an HTML page may contain. "Markup language" refers to the way tags are used to define the page layout and elements within the page.

About CSS

CSS stands for Cascading Style Sheet. Cascading style sheets are used to format the layout of Webpages

About HTTP

HTTP stands for Hypertext Transfer Protocol. HTTP is the protocol used to transfer data over the web. It's a part of the Internet Protocol suite.

About The Web

The Web is the common name for World Wide Web, a subset of the Internet consisting the pages that can be accessed by the browser
